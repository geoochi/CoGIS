CoGIS
========
本项目是中国地质大学（武汉）信息工程学院GIS工程实习题目，实现一个小型的原型系统，现将源代码开源。

关于数据
========
本项目分为两个版本，一个数据库版本，一个文件版本，这里暂时只提供文件版本。
